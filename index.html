Comandos GIT git status = gerifica status dos arquivos git log = verificas os commits feitos git init = adiciona pasta no git git add (nome do arquivo) = adiciona arquivo no git git commit = comita arquivo para dentro do git git remote = verifica se existe algum diretorio remoto git remote add (nome da origin) (endereço do git(site)) = adiciona ao git remoto git push = envia ao repositorio as alterações git push origin master = inica de onde para onde vai o repositorio git clone = clonar diretorio do git para um diretorio local git pull = puxa todas docs do diretorio
